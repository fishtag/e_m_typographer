# EMTypographer

Allows you to format yout text using http://mdash.ru API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'e_m_typographer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install e_m_typographer

## Usage

The basic usage is:

```ruby
EMTypographer.apply('supatext')

# => "<p>supatext</p>"
```
